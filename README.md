## Installation

1. Clone the repository:

```bash
git clone https://github.com/N95Ryan/lift-up.git
```

2. Install dependencies:

```bash
npx install
# or
yarn install
```

3. Start the development server:

```bash
npx expo start
# or
yarn start
```
